3. Product Decisions (Findings)

Finding 1: UX Khẩn cấp (UX Recovery)
Khi user gặp nguy hiểm, AI trả lời văn bản dài 5 dòng. Hậu quả là user chậm trễ việc cứu hộ.
Sửa bằng: Chuyển sang giao diện SOS, hiện 2 nút bấm lớn: [GỌI 113] và [GỌI AN NINH].
Finding 2: Dữ liệu Real-time (Data-tool)
Khi hỏi giá lăn bánh, AI phải đẩy sang người thật. Hậu quả là đứt gãy mạch mua sắm.
Sửa bằng: Tích hợp API tính thuế phí theo vị trí (GPS) để báo giá ngay.

4. Sketch: As-is vs To-be

As-is: User kêu cứu -> AI hiện Text dài -> User tự đọc số -> User tự thoát app gọi.

To-be: User kêu cứu -> AI hiện Warning SOS đỏ -> 1 Chạm để gọi ngay + Gửi vị trí cho cứu hộ.
